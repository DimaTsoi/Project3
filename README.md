Проект посвящен обработке данных о книгах.

Были изменены типы данных в исходном наборе данных, заполнены пробелы и удалены копии. Был проведен анализ данных, составлена описательная статистика. Визуальный анализ был проведен с использованием data lens и библиотек Яндекса: были составлены диаграммы, графический материал и матрицы.
